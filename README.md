1. Download the `48HSN_SchedulerApp.html` file
2. Open the file in your web browser
3. Start scheduling immediately - no setup required!

### Basic Usage
1. **Select Month**: Choose the month and year for scheduling
2. **Add Cohorts**: Use the dropdown management to add student cohorts
3. **Set Up Sites**: Configure clinical sites and locations
4. **Add Instructors**: Manage instructor lists
5. **Create Schedule**: Click on calendar cells to assign students, sites, and instructors
6. **Export**: Generate PDF or CSV exports when ready
