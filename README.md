A comprehensive web-based scheduling application designed specifically for nursing education programs. This application helps manage student clinical rotations, instructor assignments, and site allocations with an intuitive calendar interface.

## 🏥 Features

### Core Functionality
- **📅 Monthly Calendar Management**: Interactive calendar view for scheduling nursing students
- **👥 Student Cohort Management**: Organize students into cohorts with customizable groupings
- **🏢 Clinical Site Assignment**: Manage multiple clinical sites and locations
- **👨‍🏫 Instructor Scheduling**: Assign instructors to specific time slots and locations
- **📊 Schedule Analytics**: Built-in analytics and reporting features

### Import/Export Capabilities
- **📥 XLSX Import**: Import existing schedules from Excel files
- **📤 PDF Export**: Generate professional PDF schedules with landscape orientation
- **📄 CSV Export**: Export data for external analysis
- **💾 Auto-Save**: Automatic data persistence with session management

### Advanced Features
- **🔍 Search & Filter**: Real-time search across all schedule data
- **🎯 Conflict Detection**: Automatic detection of scheduling conflicts
- **📱 Responsive Design**: Works on desktop, tablet, and mobile devices
- **🌙 Professional UI**: Modern, clean interface with intuitive controls
- **🔐 Admin Mode**: Password-protected administrative features

### Data Management
- **📋 Dropdown Management**: Customizable lists for sites, instructors, and cohorts
- **🔄 State Persistence**: Maintains data across browser sessions
- **🧹 Data Validation**: Comprehensive validation and error checking
- **📈 Analytics Dashboard**: Visual charts and statistics

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server installation required - runs entirely client-side

### Installation
1. Download the `48HSN_SchedulerApp.html` file
2. Open the file in your web browser
3. Start scheduling immediately - no setup required!

### Basic Usage
1. **Select Month**: Choose the month and year for scheduling
2. **Add Cohorts**: Use the dropdown management to add student cohorts
3. **Set Up Sites**: Configure clinical sites and locations
4. **Add Instructors**: Manage instructor lists
5. **Create Schedule**: Click on calendar cells to assign students, sites, and instructors
6. **Export**: Generate PDF or CSV exports when ready

## 📖 User Guide

### Monthly Calendar Setup
```
1. Select month from dropdown (e.g., "NOV", "DEC")
2. Set year (default: 2025)
3. Calendar automatically generates appropriate days
```

### Adding Schedule Data
```
1. Click on any calendar cell
2. Select cohort from dropdown
3. Choose clinical site
4. Assign instructor
5. Set schedule time
6. Data saves automatically
```

### Import from Excel
```
1. Click "Import XLSX" button
2. Select your Excel file
3. Data automatically populates calendar
4. Review and make adjustments as needed
```

### Export Options
```
PDF Export:
- Professional landscape layout
- Optimized for printing
- Includes all schedule data

CSV Export:
- Complete data export
- Compatible with Excel
- Suitable for analysis
```
